Disappearing Ruby implements a cool visual feature. A combination of event listeners and event handlers gives me the ability to define how my site should react to the user's action. The reaction I have done for my site, you can trigger an event when a user hovers over Ruby like making the image disappear!
Function Name Parameters Return Value Description
toggleRuby Make image disappear and return None Event handlers that add makes the image disappear in the even the mouse goes over the image and leaves the image. If the move is illegal then nothing happens.
toggleVisibility on mouse enter and mouse leave None Changes visibilty style property from hidden to visible.
